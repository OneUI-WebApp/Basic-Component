Responsive UI란 기기의 screen size에 따라 layout, component size 등이 유연하게 변경되는 UI를 뜻한다.
Component의 margin을 지정하여 간격을 조정하거나 특정 breakpoint를 기준으로 컨텐츠, 레이아웃, 원칙 등을 지정하여 재배치할 수 있다. Responsive UI를 위해서는 하나의 기기 내에서도 screen zoom으로 변경된 최소 너비에 맞게 view 전환이 이루어지는 것을 권장한다.
본 가이드에서는 Responsive UI를 적용하기 위한 기준으로 screen width, screen height, screen orientation의 3가지 방식을 사용한다. 본 가이드에서 대표적으로 다루는 기기의 screen width와 screen height는 다음과 같다. 하기에 나열된 기기 외에 보다 다양한 기기에 대한 대응을 위해 각 component 별로 Breakpoint range가 다를 수 있으므로 보다 상세한 사항은 해당 component의 가이드 내용을 확인한다.
